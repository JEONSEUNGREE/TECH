다형성(polymorphism)
여러가지 형태를 가질 수 있는 능력
조상 타입 참조 변수로 자손 타입 객체를 다루는 것.

데이터 타입과 생성 객체가 다르다.
Ex) Tv t = new SmartTv(); //타입 불일치  ===> class SmartTv extends Tv(){}
즉 조상의 타입으로 자손객체 다룰수있다.

객체와 참조변수의 타입이 일치할때와 일치하지 않을 때의 차이
위에 예를 가져와 Tv t = new SmartTv();
조상객체 Tv에 정의된 멤버가 2개면 자손객체에 5개가 있어도 2개만 사용가능하다.
반대로 자손의 데이터 타입으로 조상 데이터타입 객체를 가르킬 수 없다. SmartTv s = new Tv(); //에러
조상보다 자손의 객체가 많은경우 자손에서 호출할경우 조상에는 없는 객체기때문에 오류가 발생한다.

결론 : 조상타입에서 자손타입 객체를 생성해야 맞는다. 자손타입 객체의 수가 많아 오류 발생할 수 있기때문에
또한 이것이 장점이다.
