[자바의 정석 - 기초편] ch7-31,32 추상 클래스, 추상 메서드

abstract class 추상클래스
-미완성 설계도. 미완성 메서드를 갖고 있는 클래스
abstract class player{
	abstract void play(int pos); // 추상메서드(몸통{}이 없는 메서드}
	abstract void stop();
}}
-다른 클래스 작성에 도움을 주기 위한 것. 인스턴스 생성 불가.
Ex) player p = new Player(); 에러 인스턴스 생성 불가

상속을 통해 추상 메서드를 완성해야 인스턴스 생성가능 (구현)
구현했을때는 abstract 제어자 제거

용도 : 꼭 필요하지만 자손 마다 다르게 구현 될 것으로 예상 되는 경우 

추상메서드 (abstract method)
-추상 메서드 호출 가능(호출할 때는 선언부만 필요)
-추상 메서드 간에 사용가능 어차피 자손클래스에서 구현할것을 예상하기때문에 구현이안되면 당연히 사용불가
